# EcoByte Design System

Design system, brand guidelines, and component library for EcoByte AS.

## Structure

```
brand/          — Logo, fonts, brand assets
docs/           — Design guide and documentation
src/            — Demo site (Astro + Tailwind)
```

## Demo Site

Live at: https://acme.ecobyte.services (behind Cloudflare Zero Trust)

## Quick Start

```bash
cd src
npm install
npm run dev
```

See [Design Guide](docs/DESIGN-GUIDE.md) for full brand guidelines including colors, typography, and tone of voice.
